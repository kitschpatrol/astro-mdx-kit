## Attributes

**Note:** Attributes are not yet supported on the Sätteri processor ([bruits/satteri#139](https://github.com/bruits/satteri/issues/139)) — the `attributes` option logs a warning and the `\{...\}` syntax is not parsed, so the example is shown as a code block:

```md
A paragraph with a class.
\{.highlight\}

![Attributed image](../../../assets/houston.webp)\{data-zoom="true"\}
```
